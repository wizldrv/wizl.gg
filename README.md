# Wizl Website

A simple "Hello World" website with a Python Flask backend and HTML/CSS/JavaScript frontend.

The website displays "wizl" in black Comic Sans font on a creamy yellow background.

## Project Structure

```
wizl/
├── app.py              # Flask backend application
├── requirements.txt    # Python dependencies
├── templates/
│   └── index.html      # Frontend HTML with styling
└── .github/
    └── copilot-instructions.md
```

## Setup Instructions

### Prerequisites
- Python 3.7 or higher
- pip (Python package manager)

### Installation

1. Install dependencies:
```bash
pip install -r requirements.txt
```

### Running the Application

Start the Flask server:
```bash
python app.py
```

The website will be available at `http://localhost:5000`

## Features

- Python Flask backend serving a simple HTML page
- Responsive design with Comic Sans typography
- Creamy yellow background (#FFFDD0)
- Black text display
